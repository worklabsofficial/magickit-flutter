# Changelog

## 1.1.0

### Added
- **MagicGridView** — Grid view component untuk layout grid responsif
- **MagicListView** — List view component dengan fitur lazy loading
- **MagicRefreshLayout** — Pull-to-refresh wrapper untuk list content
- **MagicColors** — Custom color management dengan support lerp animation
- **MagicTheme** — Enhanced theme system dengan lerp method untuk smooth transitions
- New examples dan showcase components di example app
- **MagicForm** — Async onSubmit support dengan improved child widget spacing
- **MagicDivider** — Enhanced divider dengan automatic dan adaptive spacing utilities
- **MagicText** — Enhanced text component dengan adaptive spacing

### Changed
- Enhanced MagicColors dengan custom color palette management
- Updated lerp method di MagicTheme untuk smoother transitions

## 1.0.0

- Initial stable release.
